# 第6章 数组

字符数组两种方式：

```c++
char c[3] = { 'o', 'k' }; // o, k, \0 有效长度是2
char c[3] = "ok"; // 会自动添加\0, 需要3个字节存
```

<br />

strlen: 字符数组占用的有效字节数, 不包含\0。

```c++
char c[120] = "open";
strlen(c); // 4

char c[120] = "断点abc";
strlen(c); // 2*2 + 3 = 7
```

<br />

sizeof: 定义类型的大小。

```c++
char c [120] = "open";
sizeof(c); // 120
```
